# VTT

Виртуальный игровой стол для D&D с компанией друзей. См. `CLAUDE.md` для архитектуры.

## Структура

```
backend/    FastAPI + WebSocket, состояние комнат, серверные броски
frontend/   клиент (старт — один HTML, прототип в docs/)
docs/       схемы данных и прототип UI (спецификация)
```

## Запуск (backend)

```bash
cd backend
python -m venv .venv && source .venv/bin/activate
pip install -r requirements.txt
uvicorn main:app --reload --host 0.0.0.0 --port 8000
```

`--host 0.0.0.0` чтобы друзья из Tailscale-сети достучались по `http://<твой-tailscale-ip>:8000`.

## Первая цель

Вертикальный срез: открыть прототип в двух вкладках, кинуть кубик в одной —
результат (посчитанный сервером) появляется в журнале обеих. Это поднимает
всю шину WebSocket-событий, на которую дальше навешивается остальное.
